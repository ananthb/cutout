Cutout

Open-source email proxy alias service built on Cloudflare Workers.

Built on

  • Cloudflare Workers — Rust compiled to WebAssembly
  • Cloudflare Email Routing — inbound MX rules
  • Cloudflare Email Service — outbound SMTP API
  • Cloudflare KV — rule storage and reverse-alias mappings

  • Deploy your own — fork and deploy to Cloudflare via GitHub Actions
