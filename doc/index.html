Cutout

Transparent email alias proxy — like SimpleLogin or addy.io, built entirely on Cloudflare Workers. No servers, no containers, no monthly VM bills.

How it works

  1. You own one or more domains with Cloudflare Email Routing (inbound) and Email Service (outbound)
  2. Someone emails anything@yourdomain.com
  3. Cutout worker is triggered, matches the recipient against your rules, and forwards the email to your real address
  4. A Forward rule can fan out to multiple destinations of mixed kinds: email, Telegram chat, Discord channel
  5. For email destinations:
    • Native mode preserves original From/To/DKIM/attachments (best for PGP); Reply-To is overlaid for proxying.
    • Proxy mode reconstructs the message to ensure Reply-To works when replying via the same domain (strips signatures/attachments).
    Both modes inject an X-Original-From header for permanent archival in your inbox.
  6. For Telegram/Discord: the bot posts the content to the chat; replies route back via email

Built on

  • Cloudflare Workers — Rust compiled to WebAssembly
  • Cloudflare Email Routing — inbound MX rules
  • Cloudflare Email Service — outbound SMTP API
  • Cloudflare KV — rule storage and reverse-alias mappings
